# DICOM Anonymizer

V1: This is a small tool to anonymize DICOM files and upload them to the EXACT server as nifty (.nii.gz) format.
V2: Added functionalities:
1. Go-To-Coordinate
2. Showing Informations about the file
3. Exact Coordinates
4. BUG-Fixed Y-Coordinates
